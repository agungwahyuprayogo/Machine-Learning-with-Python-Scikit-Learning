# 🤖 Machine Learning Roadmap with Scikit-Learn

Selamat datang di repositori pembelajaran Machine Learning saya! Repo ini berisi kumpulan implementasi algoritma dan teknik data science menggunakan library Scikit-Learn di Python. Isinya mencakup dasar-dasar teori hingga implementasi praktis pada berbagai jenis dataset.

## 📚 Daftar Materi & Implementasi
Saya membagi pembelajaran ini ke dalam beberapa modul utama untuk memudahkan navigasi:

| Modul | Topik            | Deskripsi Singkat                                                                   |
|-------|------------------|-------------------------------------------------------------------------------------|
| 01-04 | Dasar & Workflow | Pengenalan ML, eksplorasi sample dataset, workflow standar, dan preprocessing data. |
| 05-08 | Regression Analysis | Implementasi Simple Linear, Multiple Linear, Polynomial Regression, serta KNN Regression. |
| 09 | Data Encoding | Teknik mengubah data kategori menjadi angka (Label Encoding & One Hot Encoding). |
| 10-11 | Text Processing | Pengolahan teks menggunakan Bag of Words, Stop Word Filtering, dan TF-IDF. | 
| 12-17 | Classification Task | Berbagai algoritma klasifikasi: Logistic Regression, Naive Bayes, SVM, Decision Tree, dan Random Forest. |
